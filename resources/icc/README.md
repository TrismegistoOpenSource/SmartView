# Profilo ICC per l'OutputIntent PDF/A

`sRGB-v2-micro.icc` — profilo sRGB ICC v2 (RGB, 3 componenti) incorporato
nell'`OutputIntent` di ogni PDF/A esportato.

**Provenienza e licenza:** dai *Compact ICC Profiles* di Clinton Ingram
(saucecontrol), <https://github.com/saucecontrol/Compact-ICC-Profiles>,
rilasciati in **pubblico dominio (CC0 1.0)** — quindi liberamente
**redistribuibili** dentro l'app e dentro i file prodotti dagli utenti.

Sostituisce il precedente `sRGB.icc` copiato dal sistema macOS, che non era
licenziato per la redistribuzione. Validato conforme con veraPDF (PDF/A-2b).
